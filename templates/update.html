Update RAPs

This portable web-page uses Aruba Activate's API to quickly update RAP inventory information in bulk. Because all requests are sent directly to Aruba Activate from your browser, you must disable the cross-origin policy in your browser in order to use this page. However, please note that doing so is a security risk and is not normally recommended. It should only be used with this page and enabled again before navigating to other websites.

When submitting updates, blank entries in the table irreversibly overwrite their corresponding values in Aruba Activate. The table can also be populated by importing a CSV file. Importing from a CSV file will overwrite current entries in the table. The CSV file must be comma delimited, must contain a header row, and values must be in the same sequence as shown in the table below (i.e. MAC, Folder, AP Name, etc.). Note that the 'Controller AP Group' field is optional and is only used when generating Aruba controller whitelist commands.
